## You wanna joke well?

Joke Well is an endless well of jokes to help you tell jokes endlessly well! It uses the [Official Joke API](https://github.com/15Dkatz/official_joke_api) by [15Dkatz](https://github.com/15Dkatz) combined with advanced button technology to deliver hilarious setups and their respective punchlines with nary more than a twitch of your finger.


## Instructions

Install to your local machine, then to launch the app enter `npm run dev`

To use the cypress test suite, enter `npm run cypress` while the app is running on localhost:3000.
